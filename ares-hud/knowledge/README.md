# Cartella "knowledge"

Metti qui dentro i file (`.pdf`, `.txt`, `.md`) su cui vuoi che Ares possa
basare le sue risposte: documenti di progetto, appunti, specifiche, ecc.

Quando fai una domanda che Ares non sa gestire con le sue skill dirette
(quindi quando la passa a Gemini), cerca automaticamente nei file di questa
cartella i frammenti più pertinenti alla tua domanda e li include nella
richiesta, così la risposta può essere basata sul tuo lavoro reale.

**Privacy**: i documenti restano sempre sul tuo Mac. La ricerca nei file
avviene solo in locale — quando la domanda arriva a Gemini, gli vengono
inviati solo i pochi frammenti di testo (non l'intero file) rilevanti per
quella specifica domanda, non l'intero contenuto della cartella.

Ares rileva automaticamente se aggiungi, modifichi o rimuovi file qui dentro:
non serve riavviare il server.

Formati supportati: `.pdf`, `.txt`, `.md`.
